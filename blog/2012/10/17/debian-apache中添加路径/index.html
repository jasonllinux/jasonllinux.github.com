Debian Apache中添加路径

Oct 17th, 2012

在 /etc/apache2/con.d/目录中添加配置文件: resume.conf

内容如下：
[bash]
Alias /resume /opt/web/resume
<Directory /opt/web/resume>
Options Indexes FollowSymLinks MultiViews
AllowOverride None
Order allow,deny
allow from all
DirectoryIndex index.html
</Directory>
[/bash]

用浏览器就可以访问了 http://www.xxx.com/resume

Done!
