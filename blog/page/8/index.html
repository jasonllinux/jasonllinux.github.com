git push 错误

今天，上传代码到github上的时候 莫名其妙出现：

[bash]
Unable to rewind rpc post data - try increasing http.postBuffer
error: RPC failed; result=65, HTTP code = 401
fatal: The remote end hung up unexpectedly
fatal: The remote end hung up unexpectedly
[/bash]

解决方案：

[bash]
git config http.postBuffer 209715200
[/bash]

Debian Apache中添加路径

在 /etc/apache2/con.d/目录中添加配置文件: resume.conf

内容如下：
[bash]
Alias /resume /opt/web/resume
<Directory /opt/web/resume>
Options Indexes FollowSymLinks MultiViews
AllowOverride None
Order allow,deny
allow from all
DirectoryIndex index.html
</Directory>
[/bash]

用浏览器就可以访问了 http://www.xxx.com/resume

Done!
