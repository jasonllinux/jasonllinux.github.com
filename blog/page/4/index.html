Cocos2d-x 2.x设置竖屏方法

Sep 10th, 2012 | Comments

cocos2d-x从1.x到2.x的api转变还是有点大的，
找了半天没找到orientation的方法；
于是放狗
找到的方法貌似也不是很OK，先凑合着用
如果是ios平台
修改RootViewController.mm
[c]
(BOOL)shouldAutorotateToInterfaceOrientation:(UIInterfaceOrientation)interfaceOrientation {
return UIInterfaceOrientationIsPortrait(interfaceOrientation);

}
[/c]
如果是android平台
在AndroidManifest.xml中设置
[bash]
android:screenOrientation="portrait"
[/bash]

Archlinux之could Not Update Iceauthority File /var/lib/gdm/.iceauthority解决

Aug 31st, 2012 | Comments

早上更新了一些包之后，重启发现：

[bash]
could not update iceauthority file /var/lib/gdm/.ICEauthority
[/bash]

解决方法：

[bash]
sudo chown -R gdm:gdm /var/lib/gdm
sudo chmod 600 /var/lib/gdm/.ICEauthority
[/bash]

SVN服务器迁移

Aug 26th, 2012 | Comments

前段时间把托管比赛项目的svn服务器从自己的笔记本迁移到台式机上了，可以把所有历史记录版本信息都转走，实在是很方便：

[bash]
#备份
svnadmin dump path/to/old-repo > dumpfile

#在新的服务器上新建svn …不再赘述
#还原
svnadmin load path/to/new-repo < dumpfile
[/bash]
