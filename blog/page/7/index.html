Git Push 错误

Oct 18th, 2012 | Comments

今天，上传代码到github上的时候 莫名其妙出现：

[bash]
Unable to rewind rpc post data - try increasing http.postBuffer
error: RPC failed; result=65, HTTP code = 401
fatal: The remote end hung up unexpectedly
fatal: The remote end hung up unexpectedly
[/bash]

解决方案：

[bash]
git config http.postBuffer 209715200
[/bash]

Debian Apache中添加路径

Oct 17th, 2012 | Comments

在 /etc/apache2/con.d/目录中添加配置文件: resume.conf

内容如下：
[bash]
Alias /resume /opt/web/resume
<Directory /opt/web/resume>
Options Indexes FollowSymLinks MultiViews
AllowOverride None
Order allow,deny
allow from all
DirectoryIndex index.html
</Directory>
[/bash]

用浏览器就可以访问了 http://www.xxx.com/resume

Done!

Debian Squeeze 安装PPTP服务

Sep 26th, 2012 | Comments

入了VPS之后的第一件事情就是安装一个PPTP服务

1.安装pptpd

[bash]
sudo apt-get install pptpd
[/bash]

2.配置/etc/pptpd.conf

[bash]
##设置服务器 IP 地址与客服端 DHCP 分配地址范围
localip 192.168.0.1
remoteip 192.168.0.2-245
[/bash]

3.编辑 /etc/ppp/pptpd-options

[bash]
#为客户端提供DNS
ms-dns 8.8.8.8
ms-dns 8.8.4.4
[/bash]

4.编辑 /etc/ppp/chap-secrets

[bash]
#添加用户和密码
username   servername   password    *
[/bash]

5.编辑 /etc/sysctl.conf

[bash]
#开启IPV4转发
uncomment net.ipv4.ip_forward=1
sysctl -p
[/bash]

6.设置Iptables

[bash]iptables -t nat -A POSTROUTING -s 192.168.0.0/24 -o eth0 -j MASQUERADE

#设置MTU大小，解决部分网站无法访问的情况（MTU=MSS+20+20)
iptables -A FORWARD -p tcp –syn -s 192.168.0.0/24 -j TCPMSS –set-mss 1356
[/bash]

7.启动服务
[bash]
/etc/init.d/pptpd restart
[/bash]
