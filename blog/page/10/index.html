Cocos2d-x 2.x设置竖屏方法

cocos2d-x从1.x到2.x的api转变还是有点大的，
找了半天没找到orientation的方法；
于是放狗
找到的方法貌似也不是很OK，先凑合着用
如果是ios平台
修改RootViewController.mm
[c]
(BOOL)shouldAutorotateToInterfaceOrientation:(UIInterfaceOrientation)interfaceOrientation {
return UIInterfaceOrientationIsPortrait(interfaceOrientation);

}
[/c]
如果是android平台
在AndroidManifest.xml中设置
[bash]
android:screenOrientation="portrait"
[/bash]

Hadoop碎碎念

Hadoop configure 碎碎念，非教程！！！

1. SSH配置
免密钥

2. Host配置
User和HostName匹配

3. x.xml相关
dir配置
端口配置
java路径设置 hadoop-env.sh

4.常用操作

[bash]
#格式化
hadoop namenode -format
#启动
start-all.sh
#终止
stop-all.sh
#进程
jps
hadoop dfsadmin -report
#离开安全模式
hadoop dfsadmin -safemode leave
[/bash]

测试 启动Hadoop，
cd Hadoop目录，输入以下命令执行Pi Estimator程序：

[bash]hadoop jar hadoop-examples.jar pi 2 10
[/bash]

运行结果：

Hadoop Eclipse插件设置 Location name（取个名字）
Map/Reduce Master（Job Tracker的IP和端口，根据mapred-site.xml中配置的mapred.job.tracker来填写）
DFS Master（Name Node的IP和端口，根据core-site.xml中配置的fs.default.name来填写）

archlinux之could not update iceauthority file /var/lib/gdm/.iceauthority解决

早上更新了一些包之后，重启发现：

[bash]
could not update iceauthority file /var/lib/gdm/.ICEauthority
[/bash]

解决方法：

[bash]
sudo chown -R gdm:gdm /var/lib/gdm
sudo chmod 600 /var/lib/gdm/.ICEauthority
[/bash]
