Archlinux 安装 Virtualbox

Aug 18th, 2012

通过源安装完virtualbox之后，如果立马新建虚拟机的话会报错，

解决的方法呢？

[bash]
#查看内核版本
uname -a

[/bash]

Gentoo安装

Aug 16th, 2012

受不了Gentoo的诱惑，前端时间花了大概三天时间折腾了下Gentoo，简单记录下安装过程

[bash]
分区&&格式化
mkfs.ext4 /dev/sdaXX
mkswap /dev/sdaXX

挂载分区
sudo mount /dev/sdaXX /mnt/gentoo
mkdir /mnt/gentoo/boot
mkdir /mnt/gentoo/home
swapon /dev/sdaXX

把下载好的stageXX和portageXX解压
tar xvjpf stagexx /mnt/gentoo/
tar xvjf portageX -C /mnt/gentoo/usr

cp -L /etc/resolve.conf /mnt/gentoo/etc/

mount -t proc none /mnt/gentoo/proc
mount -o rbind /dev /mnt/gentoo/dev

chroot /mnt/gentoo /bin/bash

env-update
source /etc/profile
export PS1="(chroot)$PS1"
编辑 /etc/make.conf
+MAKEOPTS="-j5"
ACCEPT_KEYWORDS="amd64"
GENTOO_MIRRORS="http://mirrors.xmu.edu.cn/gentoo"
SYNC="rsync://mirrors.xmu.edu.cn/gentoo-portage"
VIDEO_CARDS="nvidia"
INPUT_DEVICES="evdev synapitcs"
emerge –sync
emerge gentoo-sources
emerge genkernel
zcat /proc/config.gz > /usr/share/genkernel/arch/x86_64/kernel-config
genkernel –menuconfig all

修改 /etc/fstab

修改 /etc/hostname

passwd

安装一些程序
syslog-ng mlocate dhcpcd dbus openssh alsa-utils twm xclock xterm
增加开机启动
rc-update add net.eth0 default
rc-update add syslog-ng default
rc-update add dbus default
rc-update add sshd default

安装Grub并配置
emerge grub

grep -v rootfs /proc/mounts > /etc/mtab

umount xxx && xxx && xxx

重启 应该OK了

配置桌面环境
emerge xorg-server
emerge nvidia-drivers
modprobe nvidia
update-modules
安装gdm gnome。。。。。。

添加用户
useradd -m -G uers,wheel,audio -s /bin/bash XXX
(-d /home/XX)

[/bash] 写在最后：多看Wiki，编译的想吐！！！！！！！！！！！！！！！

Linux下QtDesigner转换Pyqt代码

Aug 5th, 2012

当用Qt Designer设计Qt 界面之后，如果想使用pyqt code，就会用到 pyuic4这个命令
要使用这个命令需要先安装

[bash]
sudo pacman -S pyqt
[/bash]

装完之后，如果在你系统中有python 3.x 和python2.x共存且默认3.x的情况下，就会出错
解决方案就是切换默认python版本 最后

[bash]
pyuic4 gui.ui -o gui.py
[/bash]

Mac 安装OpenCV支援Python

Aug 4th, 2012

Mac上的OpenCV安装和Linux平台下略有不同，在这里我们不采用下载源码包手动编译的方式，因为可能有些依赖问题，我们使用Port包管理，自动安装。

Step1 安装MacPort
安装完成之后

[bash]
sudo port selfupdate
[/bash]

Step2 安装OpenCv，并且支援Python

[bash]
sudo port install py27-numpy
sudo port install opencv +python27
[/bash]

Step3 至此可能会发现python下还是不能用opencv,import 会出现错误
解决方案
切换Pyhon版本，默认使用/opt/local/bin/python27

Result 至此，应该大功告成了

黑苹果之修复linux引导

Aug 2nd, 2012

型号: thinkpad X61 OS : win7 && mac &&lion archlinux

装完mac x lion之后，linux引导是一个问题
引导程序由Grub2变成了bootthink，让我这个长期的linuxer很不适应，头可断血可流，Linux不能丢，又不能把好不容装好的99完美的mac给卸载了

天不亡我： step1 C盘根目录新建boot_linux 文件夹 step2 从linux系统的/boot/目录下,把initramfs-linux.img 和vmlinuz-linux拷贝进去 step3 在C盘根目录下新建
menu.lst
内容为:

[bash]
# general configuration:
timeout 5
default 0
color light-blue/black light-cyan/blue

# (0) Arch Linux
title Arch Linux
set root=’(hd0,10)’
kernel (hd0,0)/boot_linux/vmlinuz-linux root=/dev/sda10
initrd (hd0,0)/boot_linux/initramfs-linux.img

# (2) Windows
title Windows
rootnoverify (hd0,0)
makeactive
chainloader +1
[/bash]

Archlinux 编译 OpenCV支援Python

Aug 2nd, 2012

Step1 DownLoad OpenCV-2.4.2.tar.bz2

Step2

[bash]
cd OpenCV-2.4.2/
mkdir release
cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local -D BUILD_PYTHON_SUPPORT=ON ..
sudo make
sudo make install
[/bash]

Step3 安装完成之后会把cv的库安装到/usr/local/lib/python2.7/site-packages目录中
复制到你的python2.7所在的路径，或者加个.pth文件

或者直接修改环境变量

[bash]
export PYTHONPATH="/usr/local/lib/python2.7/site-packages/:$PYTHONPATH"
vim /etc/ld.so.conf
add "/usr/local/lib"
[/bash]

Result 进入OpenCV目录

[bash]
cd OpenCV-2.4.2/samples/python
python2.7  delaunay.py
[/bash]
