Archlinux 编译 OpenCV支援Python

Aug 2nd, 2012 | Comments

Step1 DownLoad OpenCV-2.4.2.tar.bz2

Step2

[bash]
cd OpenCV-2.4.2/
mkdir release
cmake -D CMAKE_BUILD_TYPE=RELEASE -D CMAKE_INSTALL_PREFIX=/usr/local -D BUILD_PYTHON_SUPPORT=ON ..
sudo make
sudo make install
[/bash]

Step3 安装完成之后会把cv的库安装到/usr/local/lib/python2.7/site-packages目录中
复制到你的python2.7所在的路径，或者加个.pth文件

或者直接修改环境变量

[bash]
export PYTHONPATH="/usr/local/lib/python2.7/site-packages/:$PYTHONPATH"
vim /etc/ld.so.conf
add "/usr/local/lib"
[/bash]

Result 进入OpenCV目录

[bash]
cd OpenCV-2.4.2/samples/python
python2.7  delaunay.py
[/bash]

Linux下TexLive中文配置

Jul 1st, 2012 | Comments

在Linux环境使用$$\LaTeX$$一般使用TexLive套装，而作为天朝子民，使用中文是件头等大事。

在网上大致浏览到两类方式：
第一种是盗版windows字体，修改ctex配置文件，要用中文的时候就用ctex宏包；
第二种方式就是使用XeLatex，直接调用系统中文字体。

下面直接介绍第二种方式，为了方便每次调用，我们把配置做成一个package 在该目录下新建文件

[bash]
cd /usr/local/texlive/2011/texmf-dist/tex/xelatex/fontspec
sudo touch zhfontcfg.sty
[/bash]

在该文件中写入：

[tex]
% xetex/xelatex 字体设定宏包
\ProvidesPackage{zhfontcfg}
%[cm-default]选项主要用来解决使用数学环境时数学符号不能正常显示的问题
\usepackage[cm-default]{fontspec}
\usepackage{xunicode,xltxtra}
\usepackage[BoldFont,SlantFont,CJKnumber,fallback]{xeCJK}%使用TexLive自带的xeCJK宏包，并启用加粗、斜体、CJK数字和备用字体选项
%如果没有它，会有一些 tex 特殊字符无法正常使用，比如连字符。
\defaultfontfeatures{Mapping=tex-text}
% 中文断行
%\XeTeXlinebreaklocale &quot;zh&quot;
%\XeTeXlinebreakskip = 0pt plus 1pt minus 0.1pt
%将系统字体名映射为逻辑字体名称，主要是为了维护的方便
\usepackage{indentfirst}%段落首行缩进
\setlength{\parindent}{2em}%缩进两个字符
\newcommand\fontnamehei{文泉驿正黑}
\newcommand\fontnamesong{SimSun}
\newcommand\fontnamekai{楷体}
\newcommand\fontnamemono{FreeMono}
\newcommand\fontnameroman{Times New Roman}
%%设置常用中文字号，方便调用
\newcommand{\erhao}{\fontsize{22pt}{\baselineskip}\selectfont}
\newcommand{\xiaoerhao}{\fontsize{18pt}{\baselineskip}\selectfont}
\newcommand{\sanhao}{\fontsize{16pt}{\baselineskip}\selectfont}
\newcommand{\xiaosanhao}{\fontsize{15pt}{\baselineskip}\selectfont}
\newcommand{\sihao}{\fontsize{14pt}{\baselineskip}\selectfont}
\newcommand{\xiaosihao}{\fontsize{12pt}{\baselineskip}\selectfont}
\newcommand{\wuhao}{\fontsize{10.5pt}{\baselineskip}\selectfont}
\newcommand{\xiaowuhao}{\fontsize{9pt}{\baselineskip}\selectfont}
\newcommand{\liuhao}{\fontsize{7.5pt}{\baselineskip}\selectfont}
%设置正文
\setCJKmainfont{WenQuanYi Micro Hei}
[/tex]

使用命令 生成

[bash]
sudo mktexlsr
[/bash]

测试：test.tex

[tex]
%test.tex
\documentclass{article}
\usepackage{zhfontcfg}
\begin{document}
Hello, I am jasonllinux.\
你好，我是jasonllinux
\end{document}
[/tex]

Tomcat简单配置

Jun 22nd, 2012 | Comments

在Archlinux下简单配置tomat7：

[bash]
#Install
sudo pacman -S tomcat7
#cofigure
#修改/etc/tomcat7/context.xml
<Context   reloadable="true">
#修改/etc/tomcat7/tomcat-users.xml

<!– Tomcat运行必备的角色 –>
<role rolename="tomcat"/>

<!– 进行Tomcat管理的角色 –>
<role rolename="manager-gui"/>
<role rolename="manager-script"/>
<role rolename="manager-jmx"/>
<role rolename="manager-status"/>
<role rolename="admin-gui"/>
<role rolename="admin-script"/>

<!– 进行Tomcat管理的用户，具有Tomcat管理的全部角色 –>
<user username="admin" password="apollo" roles="tomcat, manager-gui, manager-script, manager-jmx, manager-status, admin-gui, admin-script"/>
[/bash]

查看： http://localhost:8080 

Archlinux之SVN服务器搭建

Jun 20th, 2012 | Comments

最近把电脑arch化了，所以原来在Ubuntu上搭建的SVN服务也要重新安装了，貌似和Ubuntu下有些不同，参考Wiki贴上教程

1.安装Subversion和Apache

[bash]
sudo pacman -S subversion
sudo pacman -S apache
[/bash]

2.新建SVN仓库目录

[bash]
sudo mkdir -p /home/svn/repositories
[/bash]

3.修改配置文件 在/etc/httpd/conf/httpd.conf中添加
#LoadModule dav_module modules/mod_dav.so
#LoadModule dav_fs_module modules/mod_dav_fs.so
LoadModule dav_svn_module modules/mod_dav_svn.so
LoadModule authz_svn_module modules/mod_authz_svn.so
前两行不用，原配置文件中就有

SSL

[bash]
cd /etc/httpd/conf/
sudo openssl req -new -x509 -keyout server.key -out server.crt -days 365 -nodes
[/bash]

按照提示输入就成

修改/etc/httpd/conf/extra/httpd-vhosts.conf
追加
<Location /svn>
DAV svn
SVNParentPath /home/svn/repositories

AuthzSVNAccessFile /home/svn/.svn-policy-file
AuthName “SVN Repositories”
AuthType Basic
AuthUserFile /home/svn/.svn-auth-file
Satisfy Any
Require valid-user
</Location>
注意路径要一致

在/etc/httpd/conf/httpd.conf文件中加入 Include /etc/httpd/conf/extra/httpd-ssl.conf

svn-policy-file && svn-auth-file

[bash]
sudo touch /home/svn/.svn-policy-file
[/bash]

在其中添加

[/] * = r
[REPO_NAME:/] USER_NAME = rw
用你的真是仓库名替换REPO_NAME，用你的真实用户名替换USER_NAME，如果多个仓库，同理

[bash]
sudo htpasswd -cs /home/svn/.svn-auth-file USER_NAME
[/bash]

生成相应用户的密码

4.生成SVN仓库

[bash]
sudo svnadmin create /home/svn/repositories/REPO_NAME
[/bash]

设置访问权限

[bash]
chown -R http.http /home/svn/repositories/REPO_NAME
[/bash]

5.Done 大功告成，这样就可以访问了，比如 http://localhost/svn/REPO_NAME 

Reference: Archlinux Wiki 在ubuntu11.10上面安装svn服务器的过程
