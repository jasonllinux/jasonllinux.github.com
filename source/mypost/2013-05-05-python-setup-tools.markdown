# setuptools

### Install

[链接](https://pypi.python.org/pypi/setuptools#windows) 

### easy_install
* 安装一个xx包    
easy_install xx  

* 删除一个xx包  
easy_install -m xx

* 更新一个xx包  
easy_install --upgrade xx
