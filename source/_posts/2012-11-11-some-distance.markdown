记录一些平时看到的“距离”。  

#### 距离
$$distance =  \sqrt { ({ x }_{ 1 }-{ x }_{ 2 })^{ 2 } + ({ y }_{ 1 }-{ y }_{ 2 })^{ 2 }} $$

#### 曼哈顿距离
$$distance = \left| { x }_{ 1 }-{ x }_{ 2 } \right| +\left| { y }_{ 1 }-{ y }_{ 2 } \right| $$
